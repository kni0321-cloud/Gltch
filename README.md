# GLTCH Project

## Overview
GLTCH is a cyber-mysticism narrative experience exploring the boundaries between digital existence and human consciousness.

## Core Modules
- **Orb**: The central hub for communication.
- **Scan**: Reality distortion and QR code interface.
- **Sandbox**: Network visualization and node stabilization.
- **Me**: User profile and progression tracking.

## Recent Updates (v3-Final)
- **Persistence**: Enhanced state management with `gltch-v3-final` storage key.
- **Mesh Navigation**: Implemented global BottomNav with interruption support.
- **Causality**: visual consequences for SEVER actions.
